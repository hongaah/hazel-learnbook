使用

# 安装
$ npm install -g gitbook-cli

# 环境：要求低版本 node，可使用 nvm
# 使用：修改 SUMMARY.md 添加目录文件
$ gitbook init

# 输出
$ gitbook build
$ gitbook build ./ ./docs

# 开发
$ gitbook serve --port 6666
