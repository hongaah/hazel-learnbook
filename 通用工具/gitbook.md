# Gitbook

## Setup

```bash
# install
$ npm install -g gitbook-cli

# begin with nvm v10.24.1
$ gitbook install

# generate summary by gitbook-summary
$ book sm
# or
$ gitbook init

# ouput
$ gitbook build
$ gitbook build ./ ./docs

# development
$ gitbook serve --port 6666
```
