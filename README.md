## 🔗 Powered by Gitbook

```bash
# install
$ npm install -g gitbook-cli

# usage with nvm < v12
$ gitbook init

# usage
$ book sm

# ouput
$ gitbook build
$ gitbook build ./ ./docs

# development
$ gitbook serve --port 6666
```
